# Java Chat Bot

A Java-based client-server chat application with GUI support, multi-user communication, and real-time message broadcasting. This project is designed to demonstrate strong object-oriented programming (OOP) principles, networking concepts, and client-server architecture.

## Features

- Multi-user chat with server-client architecture  
- Console-based client for quick testing  
- JavaFX GUI for improved user experience  
- Threaded server to handle multiple client connections concurrently  
- Broadcast messages to all connected clients  
- Basic user session management  
- Easily extensible for authentication, encryption, and message history  

---

## Technologies

- **Language:** Java  
- **GUI:** JavaFX  
- **Networking:** TCP sockets  
- **Build Tool:** Maven (optional, recommended)  
- **IDE:** IntelliJ IDEA (recommended)
